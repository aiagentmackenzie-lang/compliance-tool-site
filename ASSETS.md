# Asset Setup

Copy your images into the `assets/` folder:

```bash
mkdir -p assets
cp /Users/main/Downloads/vonvanvillian_8K_realistic_quantum_computer_*.png assets/
cp /Users/main/Downloads/vonvanvillian_AI_Female_connected_to_an_organic_computer_*.png assets/
```

Then rename them to:
- `quantum-computer.png` - Hero background
- `project-1.png` - Security Awareness project
- `project-2.png` - Incident Response project  
- `project-3.png` - Argus Scanner project
